# Code of Conduct

We want this project to be welcoming and productive for everyone.

## Our standards

Examples of positive behavior include:

- being respectful and constructive;
- accepting feedback gracefully;
- focusing criticism on ideas and code, not people;
- helping newcomers participate.

Unacceptable behavior includes harassment, discriminatory language, personal attacks, doxxing, and sustained disruptive behavior.

## Enforcement

Project maintainers may remove comments, close issues, reject contributions, or restrict participation when behavior is inconsistent with this code of conduct.

If you need to report a conduct concern, contact the project maintainer privately.
